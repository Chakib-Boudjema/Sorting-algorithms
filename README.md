# Sorting algorithms
This repo contains known sorting algorithms implemented in c language

## Bubble Sort
time complexity : O(N^2)
## Insertion Sort
time complexity : O(N^2)
## Selection Sort
time complexity : O(N^2)
## Merge Sort
time complexity : O(N * log(N))
## Quick Sort
time complexity : O(N * log(N))
## Heap Sort
time complexity : O(N * log(N))
